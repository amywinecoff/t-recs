Install
=======================

This simulator supports Python 3.7+. It has not been tested with older versions of Python 3.

If you have not configured a Python environment locally, please follow Scipy's `instructions for installing a scientific Python distribution`_.

Currently, the simulator has only been tested extensively on MacOS 10.15.

.. _instructions for installing a scientific Python distribution: https://scipy.org/install.html

To install the simulator, you will need the Python package manager, pip_. After activating your virtual environment, run the following commands in a terminal:

..  code-block:: bash

      git clone https://github.com/elucherini/algo-segregation.git
      cd algo-segregation
      python3 -m pip install -e rec

The command will install all dependencies.

.. _pip: https://pip.pypa.io/en/stable/installing/
